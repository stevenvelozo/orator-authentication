# Orator Authentication <small>1</small>

> Server-side cookie-based session authentication with pluggable authenticator and optional OAuth/OIDC

- Pluggable username/password authenticator with session management
- Optional OAuth 2.0 / OpenID Connect login via any OIDC provider
- Microsoft-specific MSAL provider for Exchange and Graph API access
- First-class Fable service provider with zero-config defaults
